# Ecommerce com ReactJS

###  Requisitos

- (Repositorio) laravel-ecommerce-api
- Node

### Como rodar a aplicação
Execute o comando `npm start` no terminal do projeto

### Rotas da aplicação

|Rota  |Página  |
|---------|---------|
|/     |Páginia principal do aplicação         |
|/admin     |Página de administração da aplicação        |
